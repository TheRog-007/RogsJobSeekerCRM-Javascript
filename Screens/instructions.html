How To Use!

This is a port of the MS Access version including the DWP approved reports!

This application allows you to log your jobs applied for, report by sector, status and more!

Also has a mailshot facility that prints mailshot envelopes for you! CV/Letter printing is experimental as testing in live-server browser security would not allow it!

Main Forms!

This is where the information in the Data Maintenance forms comes into use

Current Forms:

- CV Template Maintenance: Not working, when is allows use to create targeted CVs for specific roles based on user selected CV elements
- Jobs Applied For:        Enter details of roles, lots of detail such as salaray and sector
- Mailshot Creator:        Create mailshots for aultiple companys and print mailing envelopes
                           CV/Covering letter printing experimental

Reports!

This is where you can analyse data entered, as the application develops more options will be added in time especially to the Job Listing report.

Current Reports:

- Job Listing Report:      See details about positions applied for DWP approved!

- Mailshot Listing Report: See contents of mailshots sent

First you need to enter data in the Data Maintenance section


For the Jobs Applied For form to work properly please visitn every form in the "Other Maintenance" section and enter data

Information in this section such as working hours (part time/fulltime) is used by the Jobs Applied For form and is required data!


Data Examples For Each Form:


- Contract Types Maintenance: Contract, permanent


- Hours Maintenance:          Full Time, Part Time


- Job Status Maintenance:     Applied For, Interview, No Response


- Work From  Maintenance:     Remote, On-site, Hybrid



Note: The CV Template Maintenance forms are implemented BUT the CV-Creator is not at this time (boo!)
